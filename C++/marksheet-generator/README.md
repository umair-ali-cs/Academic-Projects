# C++ CLI Marksheet Generator

A command-line based marksheet generator developed in C++ as an academic project.
The program allows users to generate formatted marksheets based on department and semester selection.

## Features
- Menu-driven CLI interface
- Department and semester-based subject handling
- GPA and CGPA calculation
- Text file marksheet generation
- Input validation and formatted output

## Technologies Used
- C++
- File Handling
- Arrays and Functions
- Standard Template Library (basic)

## How to Run
1. Compile the program:
   g++ marksheet.cpp -o marksheet
2. Run:
   ./marksheet

## Academic Context
Developed as a first-semester academic project at Sukkur IBA University.
